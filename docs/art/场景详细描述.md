# 游戏场景画面描述文档

## 文档说明

本文档为《心灵港湾》(Mind Beacon/Mind Haven)游戏创作，提供各场景的详细画面描述。每个场景包含多个设计方案，每个方案包括完整的画面描述、色彩方案、光影氛围、关键视觉元素及AI Prompt提示词。

**核心美术风格规范**：
- 色相：以49.7°橙黄色调为主，扩展至暖色系（橙/黄/粉/浅棕）
- 饱和度：严格控制在0.1-0.2范围（低饱和度处理）
- 明度：保持在0.8-0.9范围（高亮度表现）
- 技法：平滑渐变、细腻色彩过渡、柔和光照
- 视觉：扁平化设计、温暖亲和氛围、层次分明

### 1.1 温暖黄昏都市

**方案A1：夕阳归家**

**画面描述**：
画面呈现一座现代都市在夕阳西下时的温馨景象。地平线上，暖橙色与淡粉色交织的天空渐变为深邃的宝蓝色，夕阳悬挂天际，散发出柔和的金色光晕，将整座城市笼罩在温暖的光芒中。

城市天际线由高低错落的建筑群组成，轮廓线舒缓流畅，没有尖锐的棱角。摩天大楼的玻璃幕墙反射着夕阳的余晖，呈现出从淡金到暖橙的渐变色。较矮的建筑屋顶上，可见空调外机的剪影和晾衣架，为城市增添生活气息。

画面前景是一棵梧桐树，树叶在夕阳中呈现金橙色渐变，树影婆娑地投映在人行道上。树下的长椅上，一位老人静静地坐着，望着远方的天际线。

### 2.2 抑郁空间

**方案B2-1：暖调深海**

**画面描述**：
